What Is The Cheat List?

The Celebrity Cheat List is a playful relationship tradition where partners each choose five celebrities they jokingly agree would be an "exception." It’s meant purely for fun and conversation.

How To Use This Site

  1. Search for celebrities.
  2. Add them to Her List or His List.
  3. Each list can contain up to five people.
  4. Preview the Cheat List Agreement.
  5. Print or Save as PDF.
  6. Laugh about it together.
